Axure Utilities

从蓝湖 Axure 原型中框选提取内容，组装为结构化 Markdown 文档

🔧 Chrome Extension 📦 Manifest V3 ✓ Markdown
⚡ 快速安装
# 克隆仓库
git clone https://github.com/jove-rina/lanhu-axure-extractor-ext

# 加载扩展
打开 chrome://extensions → 开发者模式 → 加载已解压的扩展
功能

核心能力

模块化文档构建 · 智能表格识别 · 跨 iframe 拾取

🎯

框选拾取

在 Axure 页面上拖拽框选任意区域，自动识别表格或文本块，所见即所得。

🧠

智能表格识别

自动识别 Axure 标准组件，按坐标分组为行列，输出标准 Markdown 表格。

📦

模块化文档

以模块为单位组织文档，支持标题+多条内容，拖拽排序、独立展开收起。

🔄

跨 iframe 拾取

蓝湖的 Axure 内容嵌套在 iframe 中，通过广播机制实现顶层浮窗+iframe 拾取。

💾

按页自动保存

按 versionId+pageId 缓存模块数据，切换页面不丢失，重新打开即可继续编辑。

🌙

深色玻璃态 UI

全暗色主题、可拖拽浮动面板、内联 SVG 图标、中英文双语即时切换。

工作流

使用流程

三步完成从原型到文档的转换

1 登录蓝湖 → 打开 Axure 原型页面 → 点击扩展图标 → 打开文档构建器
2 新增模块 → 点击 🎯 拾取 → 在页面上拖拽框选 → 自动识别填入
3 预览确认 → 下载 Markdown 文件 → 直接用于文档、Wiki 或 AI 工具
