Axure Utilities

从蓝湖 Axure 原型中框选提取内容，组装为结构化 Markdown 文档

🔧 Chrome Extension 📦 Manifest V3 ⚡ Vite + TypeScript 🏷️ v2.0.0
⚡ 快速安装
# 从 GitHub Releases 下载预编译包
前往 GitHub Releases v2.0.0 → 下载 zip → 解压 → chrome://extensions → 开发者模式 → 加载已解压的扩展 → 选择 dist 目录
功能

核心能力

模块化文档构建 · 智能表格识别 · 跨 iframe 拾取

🎯

内容提取

拖拽框选提取页面内容；Axure 表格智能识别（按 Y/X 坐标重组）；文本降级提取；跨 iframe 拾取。

📦

文档构建

模块化标题+多条目组织；拖拽排序；独立展开/收起；全选/批量删除；聚焦模式高亮当前模块。

✏️

编辑增强

独立编辑弹窗（可拖拽/全屏）；分栏 Markdown 编辑器；撤销/重做；拾取填充条目；可折叠条目。

📤

导出

整文档/单模块：预览（深色主题 HTML）、复制、下载。文件名含真实 Axure 页面标题，日期后缀。

💾

按页保存

按 versionId+pageId 缓存模块数据至 chrome.storage.local。切换页面自动保存，关闭浏览器不丢失。

🌙

深色玻璃态 UI

全暗色主题；可拖拽浮动面板；内联 SVG 图标系统；暗色细滚动条；中/英文双语言时切换。

产品介绍

为什么需要 Axure Utilities？

让原型到文档的转换变得流畅自然

💡 产品经理、交互设计师和开发每天在蓝湖上查看 Axure 原型、对照表格、整理需求文档。Axure Utilities 将这一过程压缩为：在原型页框选区域 → 自动识别表格/文本 → 填入模块化文档 → 预览/复制/下载 Markdown。
工作流

使用流程

三步完成从原型到文档的转换

1 登录蓝湖 → 打开 Axure 原型页面 → 点击扩展图标 → 打开文档构建器
2 新增模块 → 点击 🎯 拾取 → 在页面上拖拽框选 → 自动识别填入
3 预览确认 → 下载 Markdown 文件 → 直接用于文档、Wiki 或 AI 工具
